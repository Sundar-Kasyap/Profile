## 🚀 Featured Work

### 🛰️ Bird's-Eye-View Projection — Master's Thesis
> A comparative study of deep-learning architectures for converting front-view semantic segmentation into Bird's-Eye-View, using CARLA simulation data.

I built and evaluated three models — **U-Net, U-Net + Spatial Transformer Network, and a Sequential CNN** — with a custom Dice + SSIM loss, then analyzed them quantitatively (mIoU) and qualitatively (per-scenario predictions, error maps). The interesting result: the STN improved spatial generalization and reduced false negatives, but a simpler CNN won on absolute mIoU — a clear example of a single metric hiding a real trade-off. The repo includes the honest results: what worked, what didn't, and why.

`U-Net` · `Spatial Transformer Networks` · `semantic segmentation` · `CARLA` · `TensorFlow`
